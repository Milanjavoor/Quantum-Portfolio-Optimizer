# # ⚛ Quantum Portfolio Optimizer using Qiskit

A beginner-friendly **Quantum Portfolio Optimizer** built using **Python**, **Qiskit**, and **Tkinter**. This project demonstrates how quantum computing concepts can be applied to portfolio optimization by using a QAOA-inspired quantum circuit to evaluate stock combinations based on expected return and investment risk.

Although designed as an educational project, it provides an intuitive graphical interface where users can add stocks, import portfolio data from CSV files, choose an investment profile, execute quantum-inspired optimization, and visualize measurement results through a quantum state histogram.

---

## ✨ Features

* ⚛ QAOA-inspired quantum portfolio optimization
* 📈 Calculates portfolio return, risk, and optimization score
* 🖥 Interactive Tkinter graphical user interface
* 📂 Import portfolio data from CSV files
* 💾 Export optimization results to a text file
* 📊 Quantum measurement histogram visualization
* 🎯 Multiple investor profiles:

  * Aggressive
  * Balanced
  * Conservative
* 🚀 Built using Qiskit's Aer Simulator

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries.
3. Launch the application.
4. Add stocks manually or import a CSV file.
5. Select an investor profile.
6. Click **Run Optimization**.
7. View the optimized portfolio and quantum measurement histogram.
8. Export the results if required.
